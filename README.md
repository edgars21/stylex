## Creat StyleXby creating class instance

```ts
    StyleX(element, StylexConstructor)
```

The instantiation methods accepts 2 arguments, first required HTMLElement argument
and second optional one StylexConstrcutor argument.


### StylexConstructor
Is a declerative way to define style logic and wouldm mostly likelue be used in front end declerative framewoeks like
Vue, React, Solid, Angular

`StylexConstrcutor` is a record of StylexPropertyName and StylexValue, or dynamic state record of StylexPropertyName and StylexValue
